## 🏆 Projeto em Destaque (TCC)

### [Ikommercy API](https://github.com/Sam-Umbra/dropship-final-project)
> **Engine de Backend para Dropshipping Moderno**
>
> Uma API RESTful de alta performance construída para gerenciar a complexidade de um ecossistema de dropshipping.
> * **Backend:** Java, Spring Boot, Spring Security (RBAC).
> * **Estratégia de Dados:** MySQL para persistência + Redis para versionamento de sessões em alta velocidade.
> * **Segurança:** Infraestrutura JWT com blacklisting avançado e versionamento de token.
> 
> 🔗 **[Site da Documentação](https://sam-umbra.github.io/dropship-apidocs/)** | **[Repositório Front-End](https://github.com/K4uePinheiro/Ikommercy)**
