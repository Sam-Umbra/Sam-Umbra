## 🏆 Featured Project (TCC)

### [Ikommercy API](https://github.com/Sam-Umbra/dropship-final-project)
> **Modern Dropshipping Backend Engine**
>
> A high-concurrency RESTful API built to handle the complexities of a dropshipping ecosystem.
> * **Backend:** Java, Spring Boot, Spring Security (RBAC).
> * **Data Strategy:** MySQL for persistence + Redis for high-speed session versioning.
> * **Security:** JWT with advanced blacklisting and token versioning.
>
